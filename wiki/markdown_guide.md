# Markdown Guide

### Emphasis

- **bold**
- *italics*
- ~~strikethrough~~

-----

### Headers
# Big header
## Medium header
### Small header
#### Tiny header

-----

### Lists

* Generic list item
* Generic list item
* Generic list item

1. Numbered list item
2. Numbered list item
3. Numbered list item

-----

### Links

- [Text to display](http://www.example.com)
- [Wiki Link]

-----

### Quotes

> This is a quote.
> It can span multiple lines!

-----

### Images

![](http://www.example.com/image.jpg)

-----

### Tables

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| John     | Doe      | Male     |
| Mary     | Smith    | Female   |

Or without aligning the columns...

| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| John | Doe | Male |
| Mary | Smith | Female |

-----

### Code

**Inline**

Check out this cool code: `var example = "hello!";`

**Block**

```
var example = "hello!";
alert(example);
```

**Highlighted**

```ruby
def greet(name)
    puts "Hiya, #{name}!"
end
```
